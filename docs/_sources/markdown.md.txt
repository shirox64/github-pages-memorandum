# Markdownの書き方

## 見出し
先頭に「#」をつける。「#」の後はスペースを必ず挿入する。  
```
# 見出しh1  
## 見出しh2  
### 見出しh3  
#### 見出しh4  
##### 見出しh5  
###### 見出しh6  
```

## リスト
「*」または「-」を使用する。箇条書き中はタブでインデントを下げられる。  
リストに[ ]または[x]を加えるとチャックボックスになる。
```
* リスト
* リスト
* [ ] リスト
* [x] リスト
```

## 番号付きリスト
```
1. 番号付き
2. 番号付き
3. 番号付き
```

## 改行
行末に半角スペースを2つ挿入する。
```
1行目__ ← 行末に半角スペース2つ
2行目__
```

## 空行
brタグを使用する。
```
1行目
<br>
3行目
```

## コード挿入
バッククオート3つで囲む「```」。  
結果  
```
コード
```

## リンクの挿入
```
[タイトル](URL)

(例)
[GitHub](https://github.com/)
```
結果  
[GitHub](https://github.com/)

## 引用
```
> 引用1
> > 引用2
```

## テーブル挿入
下記の書き方でテーブルが表記できる。  
2行目のコロン(:)の位置で左寄せ、中央寄せ、右寄せを指定できる。  
コロンが無ければ左寄せになる。  
```
|商品|種別|金額|
|:---|:--:|---:|
|ボンゴレビアンコ|パスタ|800|
|マルゲリータ|ピザ|1000|
|白ワイン|ドリンク|500|
```

|商品|種別|金額|
|:---|:--:|---:|
|ボンゴレビアンコ|パスタ|800|
|マルゲリータ|ピザ|1000|
|白ワイン|ドリンク|500|

## 打消し線
チルダ(~)2つで囲む。
```
~~打消されます~~
```
